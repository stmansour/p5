The XOR Neural Network

After going through all the matrix math needed to "feed-forward" inputs to the neural network to the output, and then backpropagation to adjust the weights, I thought about writing my own library for neural networks. But I opted to use Tensor Flow for many reasons. A couple of big reasons were (a) they have a bunch of functions to choose from for backpropagation of errors, (b) it's in javascript which makes it easy to write visualization graphics, and (c) it supports CPU and GPU. This is my first tensor flow project. You gotta start somewhere.
