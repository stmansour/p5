The purpose of this code is to use concepts from genetics and natural selection to solve problems that might otherwise by unsolvable in a practical sense. One example, the Infinite Monkey Theorem states that an infinite number of monkeys hitting keys on a typewriter for an infinite amount of time would eventually type any textual work you can think of, like the complete works of William Shakespear.
