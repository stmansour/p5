This tree is drawn with a recursive algorithm. The process of building it up uses the same recursive algorithm as the previous tree. In the previous example, the tree is calculated and drawn from scratch every animation frame. However, this tree's structure is built up once, as an array of Branch objects managed by a Tree object. While this can result in a very large array of objects, its advantages are that branches or points on the tree can be easily animated. Try the Wobble and Jitter effects.
